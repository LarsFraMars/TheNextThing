# Star Wars Snake

A browser-based Snake game with a Star Wars vibe.

## Run locally

Open `index.html` directly in your browser.

If you want to run a local server instead, install dependencies and start the server:

```bash
npm install
npm start
```

## GitHub backup

Når du har installert Git, kan du sikkerhetskopiere prosjektet med:

```bash
git init
git add .
git commit -m "Add Star Wars Snake game"
git remote add origin https://github.com/LarsFraMars/TheNextThing.git
git branch -M main
git push -u origin main
```

Hvis du vil at jeg skal gjøre dette automatisk når Git er installert, si fra.

## Controls

- Arrow keys or `W`, `A`, `S`, `D` to steer
- Eat power cells to grow
- Avoid hitting walls or your own tail
